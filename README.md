# react-burgers

A React-only implementation of [Johnathan Suh's Hamburgers](https://jonsuh.com/hamburgers/). No Sass or CSS required.

## Installation

With npm:

```bash
npm install --save react-burgers
```

Or with Yarn:

```bash
yarn add react-burgers
```

## Usage

```jsx
import { Slider } from 'react-burgers'

...

<Slider />
```
